AMR data 

100,000 rows of 12 variables

These data represent the sort of data that might be obtained from the 2nd generation surveillance system.

*id Integer - A unique identifier for each person 

*dob Character - a string giving the date of birth 

* spec-date Character - a string giving the date a specimen was taken 

* sex_male Binary - indicates whether the person from whom the specimen was taken was male or not. 
1 (male) 0 (not male)

*region Character - a string indicating the England region of laboratory testing the specimen 

*for person from whom specimen was taken. Range : 1 (least deprived) - 5 (most deprived)

 This data will include information about age, sex, ethnicity, country of origin
